## 📋 Инструкция для VPS деплоя:

### 1. Клонирование и установка:
```bash
git clone <repository> chiwawa-server
cd chiwawa-server
npm install
```

### 2. Настройка:
```bash
cp .env.example .env
# Редактировать .env с настройками VPS
```

### 3. База данных:
```bash
npm run db:create
```

### 4. Запуск:
```bash
npm start              # Обычный режим
# ИЛИ
npm run pm2:start      # PM2 кластер
```
